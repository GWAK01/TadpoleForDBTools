# 알림
테드폴디비허브 프로젝트의 모든 요소를 점검하고 관리하기 위해 당분간 프로젝트를 잠급니다.

오픈소스 프로젝트로 생계를 유지하는 입장에서는 여간 위협이라고 느껴집니다. 어떻게든 대응을 해야 겠기에, 하는 조취입니다. 
지금은 이전보다 소스 노출 범위를 줄이고 좀더 방어적 소극적이 될 확율이 높아 보입니다. (재 오픈은 4월 8일입니다)
